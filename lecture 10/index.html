Dispaly:flex - wrap

The CSS flex-wrap property specifies whether flex items are forced onto a single line or can wrap onto multiple lines within their container
The flex-wrap property accepts three main values:
1.nonwrap
2.wrap
3.wrap-reverce

Dispaly:flex- 1.nonwrap

This is the default value. All flex items will be on a single line, even if they exceed the container's width, potentially causing overflow.

Dispaly:flex - flex direction

The flex-direction CSS property sets how flex items are placed in the flex container defining the main axis and the direction (normal or reversed).
The flex-direction property accepts four main values:
1.row
2.row-reverce
3.colum
4.colum-reverce

Dispaly:flex- direction : 1.Row

The flex container's main-axis is defined to be the same as the text direction. The main-start and main-end points are the same as the content direction.
